# @smarttips/web

Next.js 14 (App Router) frontend for SmartTips.

## Routes
- `/` - Landing page
- `/login`, `/signup` - Auth
- `/dashboard/*` - Owner/Manager interface
- `/employee/*` - Employee personal view

## Stack
- Next.js 15 + React 19
- TypeScript strict
- Tailwind CSS + shadcn/ui
- TanStack Query for server state
- React Hook Form + Zod for forms
